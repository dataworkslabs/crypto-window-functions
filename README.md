# Cryptocurrency Price Analysis

Analysis of Bitcoin and Ethereum price trends from Jan 2022 to present using window functions in R.

## What it does
- Pulls historical price data for BTC and ETH
- Calculates year-to-date averages
- Calculates 6-day moving averages
- Compares volatility and trends

## Technologies
- R
- dplyr (window functions)
- crypto2 package
